Introduction

HTML stands for Hyper Text Markup Language, which is the most widely used language on Web to develop web pages. HTML was created by Berners-Lee in late 1991. HTML is a MUST for students and working professionals to become a great Software Engineer specially when they are working in Web Development Domain.

Here are some benefits of learning HTML:

  • Create Web site - You can create a website or customize an existing web template if you know HTML well.
  • Become a web designer - If you want to start a carrer as a professional web designer, HTML and CSS designing is a must skill.
  • Understand web - If you want to optimize your website, to boost its speed and performance, it is good to know HTML to yield best results.
  • Learn other languages - Once you understands the basic of HTML then other related technologies like javascript, php, or angular are become easier to understand.

h1 to h6 tags

The <h1> to <h6> tags are used to define HTML headings.
<h1> defines the most important heading. <h6> defines the least important heading.

Links

Links are everywhere when you surf through internet. In HTML there's thing called Hyperlink which connects us to another webpage or website.

Example

The HTML <a> tag defines a hyperlink. It has the following syntax:

<a href="url">link text</a>
Image

Image is the most important factors when it comes to style, and appearance. We could add images through image files that saved in your computer. Or we can use url. Either way it's possible to style, position, and resize by CSS.

Example

<img src="pic_trulli.jpg" alt="Italian Trulli">
Title

Have you ever saw the titles of websites on your browser? With <title> tag it's possible. With being said, it's essential thing for a website.

Example
